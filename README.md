# Customer Segmentation in Python

A portfolio-ready data science project focused on customer segmentation using Python and unsupervised learning techniques.

## Project Goal
The goal of this project is to analyze customer behavior and group customers into meaningful segments based on purchasing patterns and customer-related features.

## Tools
- Python
- pandas
- numpy
- matplotlib
- scikit-learn
- Jupyter Notebook

## Planned Workflow
1. Data loading and cleaning
2. Exploratory data analysis
3. Feature engineering
4. Data scaling
5. KMeans clustering
6. Cluster evaluation
7. Visualization and business interpretation
